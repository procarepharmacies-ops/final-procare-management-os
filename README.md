# Final ProCare Management OS

> Modern pharmacy & clinic management system built with React, Node.js, and SQL Server

## Features
- 🏥 Point of Sale (POS) with full pharmacy workflow
- 📊 AI-powered analytics dashboard
- 🗂️ Kanban board for employee task management
- 💬 WhatsApp integration for invoices & prescriptions
- 🔐 Role-based access control (RBAC)
- 💊 Titan Drug Eye substitution engine
- 📱 Prescription OCR reader
- 🏪 Multi-branch management

## Stack
- **Frontend:** React 18 + Vite + TypeScript
- **Backend:** Node.js + Express + mssql
- **Database:** Microsoft SQL Server (Mashala/Elsanta)
- **Auth:** JWT + bcrypt

## Setup
1. Clone the repo
2. `cd frontend && npm install && npm run dev`
3. `cd backend && npm install && npm run dev`
